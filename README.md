# Heather Wildfire Simulator

A Python-based wildfire simulation project that models how fire can spread through heather vegetation environments.

The project is designed as an experimental eliptical simulator for visualizing wildfire behavior and exploring how environmental factors could affect fire movement over time. As well as calculating needed amount of supressant for different supression strategies.

## Overview

Heather and moorland landscapes can be highly flammable under dry conditions. This simulator focuses on representing how fire may spread across a landscape where heather is the main fuel type.

The simulation can be used to experiment with fire spread by changing variables like wind, moisture, terrain, and vegetation.

## Features

- Simulates wildfire spread through heather vegetation
- Written in Python
- Uses a single main simulator file
- Can be expanded with more environmental variables
- Suitable for experimentation, learning, and further development
